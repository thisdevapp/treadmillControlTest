# 워터 트레드밀 컨트롤러 (Water Treadmill Controller)

ESP32-C3와 USB 시리얼 통신을 통해 워터 트레드밀을 제어하고 모니터링하는 Flutter 기반의 통합 제어 시스템입니다. 타블렛과 스마트폰 환경 모두에 최적화된 반응형 대시보드와 지능형 연결 시스템을 제공합니다.

## 🚀 주요 기능

### 1. 지능형 USB 연결 시스템
- **영구적 자동 재연결**: 마지막으로 연결된 기기의 정보(VID, PID, 장치명)를 로컬 저장소에 기억하여, 앱 재시작 시에도 별도 조작 없이 즉시 연결됩니다.
- **스마트 기기 선택 팝업**: 
    - 새로운(등록되지 않은) 기기가 연결될 때만 자동으로 기기 선택 팝업이 나타납니다.
    - 이미 등록된 기기가 연결되면 팝업 없이 백그라운드에서 자동으로 연결을 완료합니다.
- **실시간 상태 모니터링**: 2초 주기로 하드웨어 연결 상태를 감시하며, 연결 해제 시 즉각적인 시각적 피드백과 함께 복구 안내 알림을 제공합니다.

### 2. 고도화된 UI/UX 및 피드백
- **동적 상태 시각화**: 
    - 기기 미연결 시 제어부 패널이 반투명하게 비활성화되어 오조작을 방지합니다.
    - 연결 대기 중일 때 상단 USB 아이콘에 모던한 미니멀 리플(Sonar Pulse) 효과가 작동하여 연결 필요성을 직관적으로 알립니다.
- **정밀 제어 슬라이더**: 
    - **속도**: 1 ~ 5000 M/h 범위 (1 단위 정수 제어).
    - **타이머**: 1초 ~ 20분 범위 (1초 단위 정밀 제어, MM:SS 형식 표시).

### 3. 데모 시뮬레이션 모드
- **가속/감속 로직**: 스타트 버튼 클릭 시 속도가 0에서 타겟 속도까지 서서히 증가(Ramp-up)하고, 스탑 시 서서히 감소(Ramp-down)하는 실제 하드웨어 거동을 시뮬레이션합니다.
- **실시간 카운트다운**: 설정된 타겟 시간부터 실시간으로 카운트다운이 진행되며, 시간이 종료되면 자동으로 정지 프로세스가 시작됩니다.

### 4. 디버깅 및 로그 시스템
- **로컬 저장**: 모든 UART 통신 내역과 시스템 이벤트를 밀리초 단위 타임스탬프와 함께 `app_debug_log.txt`에 저장합니다.
- **인앱 로그 터미널**: 앱 내에서 실시간으로 로그를 확인할 수 있는 로그 뷰어 오버레이를 제공합니다.

## 🛠 기술 설정

### 안드로이드 설정
- **NDK 버전**: `27.0.12077973`
- **권한 및 필터**: USB 호스트 기능 활성화 및 `AndroidManifest.xml` 내 인텐트 필터 구성.

### 사용된 주요 패키지
- `usb_serial`: 핵심 시리얼 통신 라이브러리.
- `shared_preferences`: 마지막 연결 기기 정보 영구 저장.
- `path_provider`: 로컬 파일 시스템 로그 저장.
- `intl`: 타임스탬프 및 시간 포맷팅.

## 📂 로그 파일 추출 방법
분석을 위해 기기에서 로그를 추출하는 방법입니다:
1. 안드로이드 기기를 PC에 연결합니다.
2. **Android Studio** 실행 -> **Device File Explorer**를 엽니다.
3. 다음 경로로 이동합니다: `data/data/com.example.testapp/app_flutter/app_debug_log.txt`.
4. 해당 파일에서 우클릭 후 **Save As**를 선택하여 PC에 저장합니다.
